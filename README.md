# Day03 — ランダム名言メーカー

ボタン1つで名言をランダム表示できるWebアプリです。  
コピー・お気に入り保存・X共有にも対応しています。  

## 機能
- ランダムに名言を表示
- 直前と同じ名言は表示されにくい工夫あり
- コピー機能（クリップボードに保存）
- お気に入り登録（LocalStorageで保存）
- X共有ボタン（ハッシュタグ付き）

## 技術スタック
- HTML / CSS / JavaScript（Vanilla）
- GitHub Pages で公開
